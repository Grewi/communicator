id - Произвольный идентификатор сообщения. Строка. Может включать любую комбинацию цифр и латинских букв. 
    Может являться числом в 16 битном формате. Имеет значение только в связке с time. 

time - Дата генерации сообщения в UNIX формате. 

date-post - Дата отправки сообщения

right-reply - Возможность приёма ответа на сообщение. Можно сразу обозначить, что данное сообщение не требует ответа.

public - Публичность сообщения. Строго приватное, Ограниченному кругу, Публичное.

link - ссылка на данное сообщение (если есть)

type - Тип сообщения. Должен реализовывать один из множетва типов поддерживающее данным приложением. Список типов
    динамический и может произвольно дополняться.

body-type - Если тело сообщения передано отдельно, то необходимо указать ссылку для его получения. Также тело может быть
    встроино в формате base64

body - Тело сообщения. 

attach-type - Аналогично body-type

attach - Список файлов
